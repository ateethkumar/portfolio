SDN tool for OpenFlow Controllers

The link to the github site is here

The initiative of this project is to make accessibility to remote OpenFlow controllers easy. Most of the openflow controllers have its information present in json scripts. This application uses those JSON scripts and converts them into JAVA objects using ObjectMapper. We then use them as normal java objects to perform the tasks we need - create graphs from topologies, use the ip addresses, MACs to login to them remotely and various other tasks.

Features:

  • Remote connect to OpenFlow controller Floodlight presently (Will be out soon for other controllers)
  • List of OpenFlow Switches connected.
  • List of devices/MACs connected.
  • Provide innovative graph support for Topology using Jung http://jung.sourceforge.net.
  • JSON support and ObjectMapper.
  • Login support for switches directly from Topology viewer.
  • Terminal support and SSH directly using JediTerm.

Copyrights:

  • JediTerm https://github.com/JetBrains/jediterm.
  • Java Universal Network/Graph Framework (JUNG) http://jung.sourceforge.net.
  • Pty4J - Pseudo terminal https://github.com/traff/pty4j.
  • Jackson http://wiki.fasterxml.com/JacksonHome
  • org.json http://www.json.org/java/
  • Jsch http://www.jcraft.com/jsch/

Project Touring

The link to the github site is here

The initiative of this project is provide a way for people to navigate to different places in the Country/City within the least time when they are on vacation. This application will input people/user giving us the number and names of places they want to visit. The application will then calculate and show them how they should visit the places and in which order. This application utilizes the Google Maps API. We are planning to implement the Google Search integration so that we can know the availability of the places that people want to visit.

Features:
